É necessário instalar:
- flask
- flask_cors
- openpyxl

Para abrir o servidor Flask:
- caminhe até a pasta onde está o arquivo python (cliente ou produto ou vendedor)
- executar o comando python arquivo.py (arquivo = cliente ou produto ou vendedor)

Após isso, abrir o localhost dado pelo servidor e o código funcionará.
OBS: A troca de páginas não está funcionando no localhost, apenas liveserver.
OBS²: É necessário trocar os caminhos para o CSS e JS caso for usar o liveserver.
